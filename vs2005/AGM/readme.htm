A. Installation

WARNING 1: You need Allegiance standalone server installed in order for AGM.exe to start.

Nothing else to install. Just launch "AGM2.EXE"

B. How AGM works

AGM2 works like origininal AllSrvUI32.exe (aka MSAGM) from Microsoft. It (will) allows management of Allegiance games.

Unlike MSAGM, AGM2 allows more control of games.

So far, you create a game or manage existing running game(s) (if created from 'Defaultgame' (GameX registry)).

AGM2 include a log windows to display every events happening within the game(s).

AGM2 handle to 'admin chat': forcestart which allow the game owner to start a game and 'mystats' which send back to the users his current live stats. 

C. Using AGM (new in this release = blue)

Should be obvious ...Double-click on a game to see its parameters (they cant be changed). Click on a player to see his live stats in log window.
